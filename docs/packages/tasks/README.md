# @wok-cli/tasks <sub>1.0.1<sub>

This package includes a collection of general purpose sharable tasks.

## Installation

This modules requires `@wok-cli/core` as peer dependency.

```
npm i @wok-cli/core @wok-cli/tasks --save-dev
```
