# Serve Task

Node.js server as a sharable task. Uses [BrowserSync](https://www.browsersync.io/) under the hood.

## Installation

This task requires `@wok-cli/core` as peer dependency.

```
npm i @wok-cli/core @wok-cli/task-serve --save-dev
```
